Hexo+GitHub博客搭建，多终端同步方案

发表于 2017-11-01

之前搭建过一次，但是没有备份hexo设置，小白看了一圈，还是要重来，首要问题就是解决hexo文件同步，通过GitHub仓库建立两个分支，master是发布的博客文件，hexo分支放hexo文件。记录一下过程，所用系统都是windows的。

1、基础环境配置

1.1 安装node.js

在node.js官网下载并根据提示安装。

1.2 安装git

在git官网下载并根据提示安装，记得第一次安装时，有两个安装包需要安装，一个是git-lfs-windows，还有一个是git客户端。

  • 将密钥添加到GitHub账户

    • 拷贝ssh key：

    • 添加到GitHub账号：点击GitHub头像 - Settings - SSH and GPG keys - Add SHH key，在title输入密钥名称，如your_name - PC，然后在Key里粘贴刚复制的SSH Key，点击Add SSH Key。

  • 测试SSH连接

2、github配置

  • 打开GitHub创建仓库（勾选README.md选项），命名为”yourname.github.io”，如resa.github.io
  • 在项目页面，点击1commit下方的branch: master，添加hexo分支
  • 在GitHub上将hexo设为默认分支：进入博客项目仓库 - Setting - Branchs - Default branch下拉选择hexo - Update -确定
